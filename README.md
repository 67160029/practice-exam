# Simple Calculator with History

โปรเจกต์นี้เป็นส่วนหนึ่งของ **Practice Exam – Front-End Web Development**  
ใช้สำหรับฝึกเขียน HTML, CSS และ JavaScript ตามเงื่อนไขข้อสอบ

## ✨ Features
- บวก ลบ คูณ หาร (+ − × ÷)
- รองรับเลขทศนิยม
- ป้องกันการหารด้วยศูนย์
- แสดงประวัติการคำนวณ (ล่าสุดอยู่ด้านบน)
- ปุ่ม AC สำหรับ reset ค่า

## 🛠 Technologies Used
- HTML
- CSS (Grid & Flexbox)
- Vanilla JavaScript
